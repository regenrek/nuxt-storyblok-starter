## Quick start

First, install Nuxt Storyblok Starter:

```bash
git clone git@github.com:regenrek/nuxt-storyblok-starter.git my-website
```

Copy `.env.example` to `.env` and add your Storyblok [API Access tokens](/getting-started/storyblok)
